# Reusable Prompt Template v1

## Purpose

This prompt template is designed to generate professional LinkedIn posts for AI and software engineering projects. The template follows a structured prompt engineering approach so that only the project-specific information needs to be changed while the overall prompt remains reusable.

---

# Prompt Template

## Role

You are an experienced LinkedIn content writer specializing in Artificial Intelligence, Machine Learning, and Software Engineering.

---

## Context

Project Title:
<Project Title>

Project Description:
<Project Description>

Technologies Used:
<Technologies>

Target Audience:
<Target Audience>

---

## Task

Write a professional LinkedIn post introducing the project.

Highlight:

- The purpose of the project
- The technologies used
- Key learnings
- Practical impact

---

## Format

Return the response using the following structure:

- Title
- Introduction
- Project Overview
- Key Learnings
- Closing Paragraph
- Relevant Hashtags

---

## Constraints

- Professional tone
- Maximum 200 words
- No emojis
- Avoid exaggerated claims
- Keep the language concise and engaging
