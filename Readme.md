# Prompt Library v1 – Reusable Prompt Templates for LinkedIn Content Creation

A reusable prompt library designed to generate professional LinkedIn posts for AI and software engineering projects using a structured prompt engineering approach.

---

## Project Overview

Prompt engineering is more than writing individual prompts—it involves designing reusable templates that can be adapted for different tasks by changing only a few variables. This project demonstrates how a single prompt template can be reused to generate professional LinkedIn posts for multiple AI projects while maintaining consistency in tone, structure, and quality.

The template follows a structured framework consisting of:

- **Role**
- **Context**
- **Task**
- **Format**
- **Constraints**

By modifying only the project-specific information, the same template was successfully applied to five different AI projects.

---

## Objectives

- Understand the concept of reusable prompt templates.
- Design a structured prompt using the Role–Context–Task–Format–Constraints framework.
- Generate professional LinkedIn posts using a single reusable template.
- Demonstrate prompt reusability across multiple real-world AI projects.

---

## Prompt Framework

Each prompt follows the same structure:

### Role

Defines the expertise and behavior expected from the AI.

### Context

Provides project-specific information such as title, description, technologies, and target audience.

### Task

Specifies what the AI should accomplish.

### Format

Defines the required output structure.

### Constraints

Specifies writing style, tone, length, and formatting requirements.
